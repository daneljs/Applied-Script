## Övning 1 – Informationsinsamling i Linux

### Uppgift
- Skapa ett script som automatiskt samlar in grundläggande information om ett Linux-system  
- Få en överblick av systemets status och miljö  
- Träna på Linux-kommandon och enkel automatisering  

### Lösning
- Scriptet samlar in datum och tid  
- Visar systeminformation (hostname och kernel)  
- Visar aktuell användare  
- Samlar in nätverksinformation  
- Kontrollerar öppna portar  

### Resultat
- Ger en snabb översikt av systemets konfiguration  
- Kan användas som första steg vid felsökning eller säkerhetsanalys  
- Visar hur enkla script kan automatisera återkommande kontroller
